Программа состоит условно из двух частей: Сервер и Клиент.
Сервер вседа один, Клиентов может быть несколько.

Клиент:
Состоит из двух модулей: для трансляции видео с камеры (FrameCapture) и для непрерывной передачи через поток вывода
OutputStream снятых изображений. Помимо этого Клиент должен иметь возможность, до начала трансляции с камеры, принять
команду (строку) для того чтобы он начал трансляцию. Это необходимо чтобы можно было смотреть видео когда надо, а не
когда Клиент запускается на компьютере.

Сервер:
Должен принимать от Клиента (считывать потоком InputStream) входящие изображения непрерывно, так же возможно должна быть
возможность подключаться к нескольким клиентам (необходимо использование пула потоков). У сервера должна быть возможность
отправки команды (строки) через OutputStream по указанному адресу (сокету) Клиента для начала трансляции видео.
Сервер должен иметь графическую оболочку ввиде диалогового окна, где будет поле для ввода адреса компьютера где
установлен Клиент, и кнопка запуска трансляции. Трансляция должна начинаться после ввода адреса и нажатия кнопки,
при вводе нового адреса предыдущая трансляция останавливается и начинае новая в этом же окне.
